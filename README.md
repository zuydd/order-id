## 🛠️ Hướng dẫn cài đặt

> Yêu cầu đã cài đặt NodeJS

- Bước 1: Tải về phiên bản mới nhất của tool [tại đây ⬇️](https://github.com/zuydd/order-id/archive/refs/heads/main.zip)
- Bước 2: Giải nén tool

## 💾 Cách thêm dữ liệu

> Tất cả dữ liệu mà bạn cần nhập đều nằm ở các file trong thư mục 📁 `src / data`

- [user.txt](src/data/user.txt) : chứa danh sách user data mà bạn get được từ script
- [id.txt](src/data/id.txt) : chứa danh sách id telegram theo thứ tự mà bạn muốn sắp xếp, mỗi id một dòng
- [output.txt](src/data/output.txt) : chứa danh sách user data sau khi sắp xếp


## >\_ Các lệnh và chức năng tương ứng

| Lệnh            | Chức năng                                                                                                                                 |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| `npm run start` | Dùng để sắp xếp user data theo thứ tự id telegram tương ứng |

## 🕹️ Các tính năng có trong tool

- tự động sắp xếp lại user data theo danh sách id telegram cho trước
